# 电子电路基础

### 前言

作为信电学院三个专业的基础课，电基是后续电路类课程的重要基础。课程主要包括电路原理（期中前主要内容）和模拟电路（期中后主要内容，也是期末考试的大头）。

由于复习安排的问题，期末周我没有抽出足够多的时间来复习电基，~~也算是给了我一个惨痛的教训~~

因此我也打算利用假期时间好好的再学习一下这部分内容，希望能重新整理一份简练的、学习中常常忽视的讲义出来（进行中）。

这门课程的课程资料挺多的，课题组的刷题网站我个人认为还是不错的：[Fundamentals of EE Circuits](https://bc0403.github.io/EC2024/)（每年应该会更新，以当年为准）
